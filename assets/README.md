# Assets Directory

This directory contains visual assets for the ERKLIG project.

## Files

| File | Description | Dimensions |
|------|-------------|------------|
| `logo.png` | Main logo | 400x400 px ✅ |
| `erklig-banner.png` | GitHub social banner | 1280x640 px |
| `screenshot-*.png` | Screenshots | Various |

## Logo Design Suggestions

Recommended themes for the logo:

1. **Erlik Khan** - Underworld deity from Turkish mythology
2. **Shield & Sword** - Security and protection
3. **Eye** - Threat detection and surveillance
4. **Terminal/Code** - Technical tool aesthetic
5. **Wolf** - Turkish cultural symbol of strength

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Primary | `#00FFFF` (Cyan) | Logo, headers |
| Alert | `#FF0000` (Red) | Threat warnings |
| Background | `#1E1E1E` | Terminal theme |
| Success | `#00FF00` (Green) | Safe status |
| Accent | `#FF00FF` (Magenta) | Highlights |
